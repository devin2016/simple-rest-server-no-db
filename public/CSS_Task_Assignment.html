Welcome To Andreas08!

I get a log of requests about new open source templates, and this one was created as a response to one of the those requests. This template is based on the design of my own website, with only small changes (hopefully improvements!) from my own design. Among the changes, it is worth to note that this template does not use any images - and browser-based front resizing is fully supported. Other than that, there no special features or extras, only valid and well-structured XHTML 1.1 and CSS2 code.
